 Password Vault
A simple, secure password manager that stores all your passwords locally on your computer with encryption.
What It Does
Keep all your passwords in one safe place! No cloud, no internet - everything stays on YOUR machine.
How to Use
Run: python password_vault.py
Create a master password (remember this!)
Start adding your passwords
Features

 Add - Save new passwords
 View - Look up any password
 List - See all saved services
 Delete - Remove passwords you don't need
 Reset - Change your master password
 Secure - Everything encrypted with XOR + SHA-256

Forgot Password?
Type reset when it asks for master password. Warning: This deletes everything!
Files Created

passwords.json - Your encrypted passwords
master.key - Hash of your master password

Made with Python - No external libraries needed
